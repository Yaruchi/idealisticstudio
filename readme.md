# IDEALISTIC — Studio Blog Website

![IDEALISTIC Studio](logo.png)

> A minimalist blog platform for IDEALISTIC Studio, created to share development updates, project progress, announcements, and creative stories.

## ✦ About

IDEALISTIC is a custom-built studio website designed as a digital space for publishing:

- Studio blogs
- Project updates
- Behind-the-scenes content
- Community posts

The website features a clean monochrome design with interactive elements, dynamic content loading, and a responsive layout.

## ✧ Features

- Studio blog and project posts
- Dynamic team showcase
- Community feed
- Day/night theme
- Interactive hidden easter eggs
- Custom ID card generator

## Technologies

- HTML5
- CSS3
- JavaScript
- Canvas API
